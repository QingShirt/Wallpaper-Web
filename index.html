<Wallpaper Web>

Synopsis

  • <Wallpaper Web> allows you to get the system information and interaction of your local computer with your web wallpaper: open the local APP, get the volume of the computer, cpu usage......
  • No interface, boot-on, trace memory occupation

Create a connection

All functions are available in three method bodies in JavaScript:

//Create a link
WWeb = new WebSocket("ws://127.0.0.1:33001");

//Create a sending information method body
function WebSend(string){
    WWeb.send(string)
}

//Create a method body for receiving information
WWeb.onmessage = function(e) {
    Wget = eval("({" + e.data + "})");
}

Example:

/* JavaScript */

//Create a method body for receiving information
WWeb.onmessage = function(e) {
    Wget = eval("({" + e.data + "})");
    
    //Gets the current system volume value and writes to the P tag
    document.getElementById("p1").innerText = Wget.GetVolume;
    
    //Get CPU usage and write p tags
    document.getElementById("p2").innerText = Wget.GetCpuUS;
}
